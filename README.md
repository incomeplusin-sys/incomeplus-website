# IncomePlus.in - Educational Platform

## Overview
IncomePlus.in is a **stock market educational platform** providing learning tools and practice resources for fundamental and technical analysis education.

## Important Legal Notice
**This is an EDUCATIONAL PLATFORM ONLY.**
- NOT an investment advisory service
- NOT SEBI registered
- Does NOT provide stock recommendations or investment advice
- Tools are for LEARNING AND PRACTICE PURPOSES ONLY

## Website Structure
- `index.html` - Main landing page
- `style.css` - Styling
- `script.js` - JavaScript functionality
- `README.md` - This documentation

## Setup Instructions
1. Clone this repository
2. Open `index.html` in a web browser
3. For live deployment, upload to GitHub Pages or any web hosting
