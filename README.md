# IncomePlus Security Scanner Platform

A comprehensive, compliance-focused security scanning platform built with Flask, PostgreSQL, and Redis. Designed specifically for the Indian regulatory environment.

## Features

### Compliance First
- **IT Act 2000 Compliance**: Full compliance with Indian cyber laws
- **SEBI & RBI Guidelines**: Built-in checks for financial sector compliance
- **GDPR & DPDPA Ready**: Data protection and privacy compliance
- **Automatic Blacklisting**: Government/military domains blocked
- **Multi-layer Verification**: Email, phone, and identity verification

### Security Scanning
- **Website Vulnerability Scanner**: XSS, SQLi, security headers, SSL/TLS
- **SSL/TLS Analyzer**: Certificate chain analysis, protocol support
- **Port Scanner**: Ethical scanning with rate limiting
- **Custom Scanner Integration**: Easy to add your own scanners

### Business Features
- **Razorpay Integration**: Subscription management
- **Multi-tier Pricing**: Free, Pro (₹499/month), Enterprise (₹1499/month)
- **Referral System**: Bonus scans and commissions
- **Admin Dashboard**: Complete compliance monitoring
- **Audit Trail**: All activities logged for legal compliance
